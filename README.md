# 🏨 Hotel Booking System

A fully functional hotel booking web application built using **HTML, CSS, JavaScript**, with **Docker containerization**, **GitHub version control**, and **CI/CD pipeline integration**.

---

## 🚀 Features

-  User Login System
-  Room Search & Filtering
-  Check-in / Check-out Date Selection
-  Dynamic Price Calculation (based on days)
-  Room Details with Facilities
-  Payment Page (Simulation)
-  Booking Receipt Generation
-  My Bookings History
-  Responsive UI Design

---

## 🧰 Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend (Simulated): LocalStorage
- Containerization: Docker
- Version Control: Git & GitHub
- CI/CD: GitHub Actions
- Testing: JavaScript + Selenium
